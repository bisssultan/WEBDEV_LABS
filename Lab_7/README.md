# Lab 7 — Introduction to Python

## Overview

In this lab we practiced the basics of Python programming and applied object-oriented programming concepts. The work was divided into two main parts: solving coding challenges and implementing a small OOP project.

## Task 1 — Python Fundamentals

In the first part, multiple coding problems were solved to practice core Python concepts such as:

* variables and basic data types
* conditional statements (`if / else`)
* loops (`for`, `while`)
* lists and string operations
* functions

Problems were completed on three platforms:

* **informatics.msk.ru**
* **HackerRank**
* **CodingBat**

These exercises helped improve understanding of Python syntax and problem-solving skills.

## Task 2 — Object-Oriented Programming

In the second part, a small Python project was created to demonstrate OOP concepts.

A **base class `Animal`** was implemented with attributes like name, age, and color, along with methods such as `speak()` and `info()`.

Two **child classes (`Dog` and `Cat`)** were created using inheritance. Each class overrides the `speak()` method and adds its own behavior.

A separate `main.py` file creates objects from these classes, stores them in a list, and calls their methods. This demonstrates **inheritance, method overriding, and polymorphism**.

## Result

The lab provided practical experience with Python syntax, algorithmic problem solving, and basic object-oriented programming structure using modules and classes.
